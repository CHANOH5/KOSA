블럭요소 : line하나를 모두 사용하는 요소(div, h1~h6, p, table, ul, li, ol 태그들) 상하margin이 유지된다, 가로/세로 크기를 지정할 수 있음 인라인요소 : line하나를 모두 사용하지 못하는 요소(줄바꿈이 안되는 요소.. span) 상하margin이 유지되지 않는다, 가로/세로 자동설정(요소 body(태그가 열려서 닫힐때 까지)의 크기에 따라), 인라인블럭요소 : 인라인(line하나를 모두 사용하지 못하는 요소) + 블럭(가로/세로 크기 지정가능) (input, select, img, a)
